# Agrinho_2026
Herdeiros da Terra é um site desenvolvido para o Agrinho 2026, com o objetivo de fortalecer a conexão entre o campo e a cidade e valorizar a sucessão familiar no meio rural.

A plataforma apresenta uma rastreabilidade interativa dos alimentos, histórias de jovens que utilizam tecnologia no campo e uma feira virtual para divulgação de produtos da agricultura familiar.

Desenvolvido com HTML, CSS e JavaScript, o projeto demonstra como a tecnologia pode incentivar os jovens a permanecerem no campo, promovendo inovação, sustentabilidade e desenvolvimento rural.
